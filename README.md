# Quote generator 3000

This is an application made using angular that allows you to enter a quote and have it available for all to see. You can also upvote or downvote on quotes on the application


## Technologies Used
 Some of the languages used were;
     -HTML
     -TypeScript
     -JavaScript
     -CSS


##### Setup Instructions and Installation

- Install angular CLI globally
- use ng new best-quote
-ng serve to view how the app looks on your computer
